Submit a thing!

Okay so, I don't have a formal process or a nice form or anything yet, but that doesn't mean I don't take submissions! If you're familiar with GitHub, the easiest/fastest way to submit things is by forking this project and submitting a pull request to add your YAML file to an appropriate folder in images/ - to do so, fork the project with the button in the upper-right, which will create a copy of the repo under your account. Then you can add files right in the GitHub web interface with the "Add File" dropdown, and create a pull request to merge your changes into the main repo.

If you're not familiar with GitHub and that all sounds like nonsense, no worries! You can always just email me the info as well and I can get it set up. Or if you want to learn about how GitHub works, shoot me an email or a DM or whatevs.

In either case, you can see an example of the info I need (and the format of the YAML if you're doing a Pull Request) by looking at this commit on GitHub. Namely, I need:

  • Category (TV show, book, fictional universe, etc)
  • Name
  • Credit and source URL for the image
  • One of height/width/length of the thing (in meters)
  • An image of the thing, preferably cropped and cut out with a transparent background
  • Optionally, a short blurb/other descriptive info. I don't have a place for descriptions yet, but it's good to plan ahead!
If the image isn't cut out I can probably do it myself, but it'll take me longer to get around to it.

Also, I of course reserve sole discretion in what I do and don't put up - I don't anticipate having to reject any well-intentioned submissions, but this is the internet, so heaven knows what y'all will come up with.

Thanks for reading, and if you have any questions, feel free to let me know!
